Цепочка промисов

Создай 3 функции:
  step1 — возвращает промис, который через 500 мс резолвится строкой "Шаг 1"
  step2 — принимает строку, добавляет " -> Шаг 2", возвращает промис через 500 мс
  step3 — аналогично добавляет " -> Шаг 3" и возвращает промис через 500 мс

Собери цепочку:
step1()
  .then(result => step2(result))
  .then(result => step3(result))
  .then(result => console.log(result)) // Должно быть: "Шаг 1 -> Шаг 2 -> Шаг 3"
