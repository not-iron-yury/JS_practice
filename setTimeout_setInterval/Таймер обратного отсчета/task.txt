Задача 8: Таймер обратного отсчета

Контекст
Представь, что ты разрабатываешь модуль для интернет-магазина.
Нужно создать ограниченный таймер (например, для отображения таймера скидки на товар).

Требования к функции:
1. Таймер должен запускаться с определённым числом секунд.
2. Таймер должен уменьшаться каждую секунду.
3. Должен быть метод для остановки таймера.
4. Должен быть метод для получения оставшегося времени.
5. После завершения таймера, он не должен уменьшаться дальше нуля.

Входные данные:
Функция createCountdownTimer(duration) принимает:
duration (число) — количество секунд.

Возвращает объект с методами:
start() — запускает таймер.
stop() — останавливает таймер.
getTimeLeft() — возвращает оставшееся время.

Пример использования

const timer = createCountdownTimer(5);

timer.start();
setTimeout(() => console.log(timer.getTimeLeft()), 3000); // 2 (через 3 секунды)
setTimeout(() => timer.stop(), 3000); // Остановка на 4-й секунде
setTimeout(() => console.log(timer.getTimeLeft()), 6000); // 2 (таймер не уменьшился после stop)

Подсказка
Используй setInterval, а для хранения состояния таймера — замыкание.
