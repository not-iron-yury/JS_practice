Типовые сценарии использования замыканий

1.Создание приватных переменных
→ Пример: Инкапсуляция состояния внутри модуля или компонента.

2. Фабрики функций
→ Пример: Генерация обработчиков событий или динамических функций.

3. Оптимизация производительности (декораторы, кеширование, мемоизация)
→ Пример: Функции debounce и throttle.

4. Обработчики событий (event listeners)
→ Пример: Сохранение данных между вызовами обработчика.

5. Замыкания в асинхронных операциях (setTimeout, setInterval, промисы)
→ Пример: Захват контекста для использования в callback'ах.

6. Скрытие реализации (паттерн модуль)
→ Пример: Создание IIFE (Immediately Invoked Function Expression).

7. Формирование частично применённых функций (partial application, curry)
→ Пример: Функции с предустановленными аргументами.
