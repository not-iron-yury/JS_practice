Задача 5: Запоминание результатов (Memoization)

Контекст
Ты разрабатываешь функционал для кеширования результатов вычислений,
чтобы ускорить выполнение функции при повторных вызовах с одинаковыми параметрами.

Напиши функцию memoize(fn), которая:
1. Принимает функцию fn (например, вычисление какого-то значения).
2. Возвращает новую функцию, которая будет запоминать (кешировать) результаты предыдущих вызовов.
3. Если функция вызывается с теми же аргументами, она должна возвращать кешированное значение вместо того, чтобы вычислять его заново.

Пример использования
function slowFunction(a, b) {
  console.log("Выполняется вычисление...");
  return a + b;
}

const memoizedFunction = memoize(slowFunction);

console.log(memoizedFunction(2, 3)); // Выполняется вычисление... 5
console.log(memoizedFunction(2, 3)); // 5 (из кеша)
console.log(memoizedFunction(3, 4)); // Выполняется вычисление... 7

Ограничения:
1. Нужно сохранять кеш, но только для тех значений, которые уже вычислялись.
2. Каждый уникальный набор аргументов должен кешироваться.
