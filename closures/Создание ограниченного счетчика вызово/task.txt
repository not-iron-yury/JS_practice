Задача 4: Создание ограниченного счетчика вызовов

Контекст
Ты работаешь над системой защиты от спама. Нужно создать функцию, которая ограничивает количество вызовов другой функции.

Напиши функцию createLimitedFunction(fn, limit), которая:
1. Принимает функцию fn и максимальное количество вызовов limit.
2. Возвращает новую функцию, которая вызывает fn, но не чаще, чем limit раз.
3. После превышения лимита должна просто возвращать undefined без выполнения fn.

Пример использования
function sayHello() {
  console.log("Привет!");
}

const limitedHello = createLimitedFunction(sayHello, 3);

limitedHello(); // "Привет!"
limitedHello(); // "Привет!"
limitedHello(); // "Привет!"
limitedHello(); // (ничего не происходит)
limitedHello(); // (ничего не происходит)

Ограничения:
1. Количество оставшихся вызовов должно храниться в замыкании.
2. Внешний код не должен иметь доступа к этому значению.
