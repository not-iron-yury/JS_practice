Задача 1: Создание приватных переменных

Контекст
Ты разрабатываешь UI-компонент для интернет-магазина — "Счетчик товаров".
У пользователя есть кнопки "Добавить" и "Убрать", но нельзя допускать, чтобы количество товаров становилось меньше 0.

Нужно написать функцию createCounter, которая создает счетчик с приватным состоянием. Он должен иметь два метода:
  increment() — увеличивает количество товаров на 1.
  decrement() — уменьшает, но не позволяет уйти в отрицательные значения.
  getCount() — возвращает текущее количество товаров.

Пример использования

const counter = createCounter();

counter.increment();
counter.increment();
console.log(counter.getCount()); // 2

counter.decrement();
console.log(counter.getCount()); // 1

counter.decrement();
counter.decrement();
console.log(counter.getCount()); // 0 (не уходит в отрицательные значения)

Ограничения:
1. Внешний код не должен иметь доступа к изменению состояния напрямую (count должен быть приватным).
2. API должно быть удобным и простым в использовании.
